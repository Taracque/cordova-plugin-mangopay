# cordova-plugin-mangopay

A Cordova plugin to use Mangopay Client SDKs in Cordova projects

Api reference:

```registerCard(options, onSuccess, onError)```

options {
    cardRegistrationURL
    preregistrationData
    accessKey
    clientId
    baseURL
    cardPreregistrationId
	cardNumber
    cardExpirationMonth
    cardExpirationYear
    cardCvv
}
